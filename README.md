# ProtoSDL3Ando

Projet prototype utilisant SDL3 en C++.

## Prérequis

- CMake 3.20 ou supérieur
- SDL3 installé sur votre système
- Compilateur C++17 (GCC, Clang, ou MSVC)

## Compilation

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Exécution

```bash
./ProtoSDL3Ando
```

## Structure du projet

- `main.cpp` - Point d'entrée principal
- `CMakeLists.txt` - Configuration CMake
